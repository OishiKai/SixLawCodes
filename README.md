# SixLawCodes

六法全書の内容を手軽に確認できるアプリです。
e-GOV法令の法令APIをを使用しています。(https://elaws.e-gov.go.jp/apitop/)
現在以下の法律に対応しています。

- 日本国憲法 (昭和二十一年憲法)
- 刑法 (明治四十年法律第四十五号)
- 民法 (明治二十九年法律第八十九号)

## Installation

$ git clone https://github.com/oishikai/SixLawCodes

## Usage

Run iOS Simulator or iOS Devices<BR>iOS 13.0 and Later
